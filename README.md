# Expense Tracker

A web-based Expense Tracker application developed using Java, JSP, Servlets, MySQL, HTML, CSS, and JavaScript. The application helps users efficiently manage their income, expenses, and monthly budgets while providing an admin dashboard for monitoring and analytics.

## Features

- User Registration & Login
- Add, Edit, Delete Income
- Add, Edit, Delete Expenses
- Transaction Category Management
- Monthly Budget Tracking
- Expense Analytics
- Search & Filter Transactions
- Admin Dashboard
- Secure Authentication

## Tech Stack

- Java
- JSP & Servlets
- HTML
- CSS
- JavaScript
- MySQL
- JDBC
- Apache Tomcat

## Installation

1. Clone the repository:

```bash
git clone https://github.com/anushakhodike-132/ExpenseTracker.git
```

2. Import the project into Eclipse or VS Code.

3. Configure Apache Tomcat.

4. Import the MySQL database.

5. Update database credentials in the JDBC connection file.

6. Run the project on Tomcat.
